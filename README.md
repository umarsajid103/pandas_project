# Pandas CSV Reader & Basic Analysis

## 📖 Overview
This project demonstrates a basic data analysis workflow using Python and Pandas. It covers loading CSV data, inspecting the dataset, computing summary statistics, filtering and slicing data, visualizing results, and exporting processed output.

## 🎯 Features
- Read CSV into Pandas DataFrame  
- Inspect data using head, tail, and info  
- Compute summary statistics  
- Filter rows and select columns  
- Slice dataset subsets  
- Visualize salary distribution  
- Export filtered data to CSV  

## 🛠️ Tech Stack
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook
